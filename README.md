# OpenTelemetry JS Collector 
[OpenTelemetry](https://github.com/open-telemetry) missing collector server 
implementation.

This project is a PoC 

## Getting Started

### Installation

```shell
npm install
```

### Run the service 
Run the collector 
```shell
node server.js
```

run the test app
```shell
node test_app.js
```
